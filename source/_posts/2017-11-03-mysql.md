# MYSQL安装
- ubuntu 环境下面 mysql安装
```
    # 确认版本信息, 如果没有sudo apt-get update
    apt-cache search mysql 
    
    sudo apt-get install mysql-server
    sudo apt-get install mysql-client
    sudo apt-get install libmysqlclient-dev
```

# 常用命令和SQL语法
- 进去mysql终端命令为:
```
    mysql -u root -p
```
