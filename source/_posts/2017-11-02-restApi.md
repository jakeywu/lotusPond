### API版本控制方式
1. 在URL上面区分 /api/v2
```
    https://127.0.0.1/test
```

2. URL里面添加
```
    https://127.0.0.1/test?v=2
```

3. 用户自定义的request Header
```
    https://127.0.0.1/test  
    api-version: 2
```
<!-- more -->

4. 利用ContentType
```
    https://127.0.0.1/test
    Accept: application/json; version=2.0  
```
