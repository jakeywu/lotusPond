矩阵乘法 A.dot(B)
第一个矩阵第一行的每个数字（2和1），各自乘以第二个矩阵第一列对应位置的数字（1和1），然后将乘积相加（ 2 x 1 + 1 x 1），得到结果矩阵左上角的那个值3。 也就是说，结果矩阵第m行与第n列交叉位置的那个值，等于第一个矩阵第m行与第二个矩阵第n列，对应位置的每个值的乘积之和
```
[[1, 1], [0, 1]]   [[2, 0][3, 4]] == [[1*2+1*3, 1*0+1*4], [0*2+1*3, 0*1+1*4]] 
```
